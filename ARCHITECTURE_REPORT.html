Architectural Design Principles
Dual-Mode Agents
All cloud-dependent agents have rule-based fallbacks. System works without Bedrock/VoyageAI credentials - critical for hackathon resilience.
Parallel Stage 1
Fraud + Policy + Ring detection run in asyncio.gather() simultaneously. This is what enables the under-3-second end-to-end target.
Stateful Fraud Rings
Ring detection writes to MongoDB and learns patterns in agent_memory - cross-session, persistent fraud intelligence that improves over time.
Immutable Audit Trail
audit_trail collection is insert-only - never updated or deleted. Satisfies IRDAI traceability with tamper-evident records.
Human-in-the-Loop
Underwriting has explicit human checkpoints. Adjuster override creates immutable audit entry. AI recommends, human decides.
MongoDB-Backed State
LangGraph state persisted in langgraph_checkpoints - enables pause/resume for long-running flows and crash recovery.
